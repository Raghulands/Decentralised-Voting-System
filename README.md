# 🗳️ Decentralised Voting System  
### Blockchain-Based Secure E-Voting with Facial Recognition

## 📌 Overview

The Decentralised Voting System is a secure and transparent digital election platform built using blockchain technology and modern facial recognition authentication. The system leverages Ganache to simulate a blockchain environment where each vote is stored as an immutable transaction, ensuring transparency, security, and trust in the electoral process.

This project eliminates centralized control over voting data, prevents vote tampering, and strengthens voter authentication using biometric verification.

## 🚀 Key Features

### 🔗 Blockchain-Based Vote Storage (Ganache)
- Votes are recorded as blockchain transactions  
- Immutable ledger ensures tamper-proof storage  
- Transparent and verifiable vote records  
- Prevents duplicate or altered votes  

### 🧠 Facial Recognition Authentication
- Biometric verification before vote casting  
- Prevents identity fraud and impersonation  
- Ensures one-person-one-vote principle  

### 🔐 Multi-Layer Security
- Secure authentication mechanisms  
- Encrypted vote transmission  
- Blockchain-backed data integrity  
- Transaction validation before confirmation  

### 🏛️ Admin & Candidate Management
- Candidate registration and approval  
- Election monitoring dashboard  
- Real-time vote tracking  
- Transparent result generation

## 🧩 System Architecture

1. User registers and verifies identity through facial recognition  
2. Verified user logs into the system  
3. User casts vote through secure interface  
4. Vote is converted into a blockchain transaction  
5. Ganache records the transaction immutably  
6. Results are computed transparently from blockchain records  

This distributed approach removes reliance on centralized vote storage and enhances system trust.

## 🎯 Problem Statement

Traditional voting systems face challenges such as centralized control, vote manipulation risks, identity fraud, and lack of transparency. This project demonstrates how blockchain and biometric authentication can modernize digital elections while preserving voter privacy and data integrity.
